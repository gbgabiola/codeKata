# Level 3


```js
var x = [1, 3, 5];
x[0] = x[2];
x[1] = x[0]
console.log(x);
```

| Variable |           Values           |
| :------: | :------------------------: |
|   `x`    | `[1, 3, 5]` -> `[5, 5, 5]` |

1. **Answer**: `[5, 5, 5]`
